Quick Links

  • /tags - add a new tag(login required) and list existing tags

  • /docs - add a new doc(login required) and list existing docs
  • /doc/7 - show details of doc #7
  • /doc/recent?begin=2&end=5 - order all docs by id desc and show the 2nd doc to the 5th doc
  • /doc/4/upvote - voteup doc #4
  • /doc/5/downvote - votedown doc #5
  • /imgs - show image list under img
